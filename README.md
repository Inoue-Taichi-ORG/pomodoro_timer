# pomodoro_timer
現 Masterは1号機
  
## 仕様  
- 25分の集中と5分の休憩を基本セットとする
- 但し4セット目の休憩は30分と長めの休憩
- 途中でタイマーを止めることは許されない(そもそもポモドーロシステムとはそういうもの)
  

## 備えている機能  
Startボタン
- 押すとポモドーロタイマーが始まる
  
Clearボタン
- 押すとポモドーロタイマーが初期状態に戻る
